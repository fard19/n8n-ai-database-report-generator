# AI Database Report Generator

An n8n-powered AI workflow that automatically converts database assignment requirements into a complete professional database report.

## Features

* AI-powered assignment analysis
* SQL schema generation
* MongoDB script generation
* ERD generation (Mermaid)
* Database design documentation
* HTML report generation
* Automatic PDF generation using PDF.co
* End-to-end workflow automation using n8n

## Workflow

Assignment Input → Analyzer Agent → SQL Generator → MongoDB Generator → ERD Generator → Report Generator → HTML Generator → PDF Generator

## Technologies

* n8n
* Google Gemini
* PDF.co API
* MongoDB
* SQL
* Mermaid ERD

## Sample Output

The workflow generates:

* Database design documentation
* SQL DDL and DML scripts
* MongoDB implementation scripts
* ERD diagrams
* Relationship explanations
* Professional PDF reports
